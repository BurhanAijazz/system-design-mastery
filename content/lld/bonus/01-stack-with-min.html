Design: Stack with Min - O(1)

Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

Requirements

  • push(val) - Push element onto stack
  • pop() - Remove top element
  • top() - Get top element
  • get_min() - Retrieve minimum element in O(1)

Approach: Auxiliary Min Stack

Maintain a second stack that tracks the minimum at each level.

class MinStack:
    def __init__(self):
        self._stack = []
        self._min_stack = []

    def push(self, val: int) -> None:
        self._stack.append(val)
        current_min = min(val, self._min_stack[-1] if self._min_stack else val)
        self._min_stack.append(current_min)

    def pop(self) -> None:
        self._stack.pop()
        self._min_stack.pop()

    def top(self) -> int:
        return self._stack[-1]

    def get_min(self) -> int:
        return self._min_stack[-1]

Key Takeaways

  • Space-time trade-off: O(n) extra space for O(1) min retrieval
  • Each min_stack entry captures the min at that stack depth
  • Pop operations maintain correctness automatically
