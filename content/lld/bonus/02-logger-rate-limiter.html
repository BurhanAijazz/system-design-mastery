Design: Logger Rate Limiter

Design a logger that limits duplicate log messages within a time window.

Implementation

class LoggerRateLimiter:
    def __init__(self, window_seconds: int = 10):
        self._window = window_seconds
        self._timestamps = {}  # message -> last_printed_timestamp

    def should_print(self, timestamp: int, message: str) -> bool:
        if message not in self._timestamps:
            self._timestamps[message] = timestamp
            return True
        if timestamp - self._timestamps[message] >= self._window:
            self._timestamps[message] = timestamp
            return True
        return False

Key Takeaways

  • Simple HashMap-based deduplication
  • O(1) per operation
  • Consider cleanup of old entries for long-running systems
