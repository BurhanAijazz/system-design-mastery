Project: ATM Machine

A full LLD implementation of an ATM (Automated Teller Machine) system. This project uses State for ATM states, Chain of Responsibility for transaction validation, Command for transactions as undoable operations, and Strategy for cash dispensing algorithms.

Requirements

Functional:

  • Insert card, validate PIN (3 attempts max, then retain card)
  • Balance inquiry
  • Cash withdrawal (with denomination calculation to minimize notes)
  • Cash deposit
  • Transfer between accounts
  • Print receipt
  • Card ejection

Out of scope: Multi-bank networking, check deposits, bill payments, physical hardware simulation, encryption.

Design Patterns Used

  • State -- The ATM transitions through: Idle -> CardInserted -> PINEntered -> TransactionSelected -> Processing -> Dispensing -> Idle. Each state class defines what operations are valid. Idle only allows insertCard(). CardInserted only allows enterPIN(). Invalid operations raise clear error messages
  • Chain of Responsibility -- Transaction validation chain: AuthHandler (verify PIN) -> BalanceCheckHandler (sufficient funds?) -> WithdrawalLimitHandler (within daily limit?). Each handler passes or rejects
  • Command -- Each transaction (WithdrawCommand, DepositCommand, TransferCommand) implements a Transaction interface. This enables transaction logging and potential reversal
  • Strategy -- Cash dispensing algorithm: GreedyDispenser uses the largest denominations first to minimize the number of bills, checking available stock. If exact amount cannot be dispensed, it raises an error

Key Components

# BankingService interface -- abstracts the bank backend
class BankingService(ABC):
    @abstractmethod
    def verify_pin(self, card_number, pin) -> bool: pass

    @abstractmethod
    def get_balance(self, account_id) -> float: pass

    @abstractmethod
    def withdraw(self, account_id, amount) -> bool: pass

    @abstractmethod
    def deposit(self, account_id, amount) -> bool: pass

    @abstractmethod
    def transfer(self, from_id, to_id, amount) -> bool: pass


# CashDispenser manages denomination stock
class CashDispenser:
    def __init__(self):
        self._stock: Dict[CashDenomination, int] = {
            d: 100 for d in CashDenomination
        }

    def dispense(self, amount: int) -> Dict[CashDenomination, int]:
        # Greedy: largest denominations first
        result = {}
        remaining = amount
        for denom in sorted(CashDenomination,
                            key=lambda d: d.value, reverse=True):
            count = min(remaining // denom.value, self._stock[denom])
            if count > 0:
                result[denom] = count
                remaining -= count * denom.value
        if remaining > 0:
            raise ATMError("Cannot dispense exact amount")
        return result
